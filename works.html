Project 3: R3play Web API

In this project I created a Web API and linked it to a database. Users can register/ delete account, login, and search for information about movies, tv series, anime, directors, genres, add/delete a movie to/from a list of favorites and more. The project is built using Node.js, Express and other required dependencies, is hosted on Heroku and uses Mongo DB as the database.

The project's aim was to practice creating an API, working with Node.js, Express, working with SQL and NoSQL databases, writing sql querries, working with Postman, writing documentation, deploying a website to Heroku and the database to Mongo DB etc

Project 5: Meet App

In this project I created a Progressive Web App with an OAuth2 authorization and authentication that fetches events from a Google Calendar Events API. The users can search for a city, specify the number of events, see the events details, use the app both online and offline, see visualized graphs about the events information, get notifications if they are offline and/or enter incorrect data, install the app on mobile/desktop and more. The project is built with React.js, PWA requirements, Google Calendar API as a dependency, is responsive and deployed to Github pages.

This project aimed to practice using serverless AWS Lambda functions, React components, turning an app to a Progressive Web App, Object-Oriented vs Functional programming, configuring OAuth2, and most of all, unit, integration, user-acceptance and end-to-end testing.

Project 3: R3play Web API

In this project, I created a Web API and linked it to a database. Users can register/ delete accounts, log in, and search for information about movies, TV series, anime, directors, and genres, add/delete a movie to/from a list of favorites and more. The project is built using Node.js, Express, and other required dependencies, is hosted on Heroku, and uses Mongo DB as the database.

The project aimed to practice creating an API, working with Node.js, and Express, working with SQL and NoSQL databases, writing SQL querries, working with Postman, writing documentation, deploying a website to Heroku and the database to Mongo DB, etc. Additionally, the project aimed to teach how to create documentation with JSDOC

Project 5: Meet App

In this project, I created a Progressive Web App with an OAuth2 authorization and authentication that fetches events from a Google Calendar Events API. The users can search for a city, specify the number of events, see the details of the events, use the app both online and offline, see visualized graphs about the events information, get notifications if they are offline and/or enter incorrect data, install the app on mobile/desktop and more. The project is built with React.js, Progressive Web Applications requirements, and Google Calendar API as a dependency, is responsive, and deployed to Github pages.

This project aimed to practice using serverless AWS Lambda functions, React components, turning an app into a Progressive Web App, Object-Oriented vs Functional programming, configuring OAuth2, and most of all, unit, integration, user-acceptance and end-to-end testing. Also, the project aimed to practice working with user-stories.

Project 6: OWrite React Native App

In this project, I created a React Native application that allows users to log in, chat, send/take pictures, send/play audio recordings, and share geolocation information. Users can choose a username and set a chat screen background color. All user data gets stored in Firebase Firestore. If offline, users can access the chat history saved to local memory. Additionally, the project was developed with the accessibility considerations in mind.

This project aimed to practice creating React Native apps for Android and iOS and testing a native app in Android Studio/ XCode. Additionally, the project aimed to practice working with user-stories.

Project 7: MyFlix Angular App

In this project, I created a front-end part for the R3play Web API with Angular, Typescript, and Material UI. Users can register, log in, see all movies and related movie information, access the profile page, delete the account, log out, add/delete movies to/from the list of favorites, etc.

This project aimed to practice creating Angular apps for the front-end part using Material UI and Typescript. Additionally, the project aimed to practice creating Kanban boards, working with user-stores and user flows, working in a team and preparing the code to being handed off, including writing documentation. The Kanban board is important for the team to know who is responsible for what tasks, how much time and effort they require, etc. I practiced writing the tasks to complete, then moved them from "to-do", to "in-progress", and then to "done". Developing soft skills was among the project's tasks as well.
